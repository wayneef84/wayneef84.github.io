# Required Root Files
**Last Updated:** 2026-02-05

The following files **MUST** remain in the repository root for agent operations, CI/CD, and protocol adherence. **Do not move these to subfolders.**

*   `ONBOARDING.md`: Primary entry point for all agents.
*   `CLAUDE.md`, `GEMINI.md`, `JULES.md`: Agent identity files.
*   `AI_FEEDBACK.md`: Shared memory log.
*   `PROJECTS.md`: High-level registry.
*   `README.md`: GitHub display.
*   `.gitignore`: Git configuration.
*   `AGENTS.md`: Protocol definitions.
*   `CHANGELOG.md`: Master history.
*   `TODO.md`: High-level tasks.
*   `LICENSE`: Legal.
