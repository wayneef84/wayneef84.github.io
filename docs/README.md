# Documentation Library

Welcome to the central documentation hub for the Fong Family Arcade platform.

## Projects

### Games
*   [Snake (Neon Serpent)](projects/SNAKE.md): A reflex-based arcade game with synthesis audio and touch controls.
*   [Flow Games](projects/FLOW.md): A suite of path-finding puzzle games with a modular grid engine.
*   [Card Games](projects/CARDS.md): A collection of card games (Blackjack, Euchre, War) sharing a DOM-based engine.
*   [Slots](projects/SLOTS.md): A 3D-simulated slot machine with particle effects and theming.
*   [Magic XTC Ball](projects/XTC_BALL.md): A fortune-telling app with SVG graphics and vibration feedback.
*   [Sprunki](projects/SPRUNKI.md): An interactive music mixer with drag-and-drop mechanics.

### Shared
*   [Shared Utilities](projects/SHARED.md): Common libraries and assets used across the platform.
