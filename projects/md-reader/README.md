# Markdown Reader (Docs & Index)

**Policy Classification:** Personal / Non-Commercial
**License:** MIT

This utility parses and renders Markdown files from the repository, including syntax highlighting and search capabilities.

## ⚖️ Legal & Compliance

For details on data usage, privacy, and terms of service, please refer to the centralized documentation:
*   [**Legal & Policy Framework**](../../legal/README.md)
*   [**Data Policy**](../../legal/DATA_POLICY.md)

## Usage
Run `python build_index.py` to regenerate `repo_index.js`, which is used by the search functionality.
