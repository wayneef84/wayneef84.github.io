# üç≥ Fong Family Cookbook

Welcome to the digital cookbook. This is powered by the Markdown Reader's special "Cookbook Mode".

To add a new recipe, simply create a markdown file anywhere in the repository (e.g., `recipes/pasta.md`) and make sure it has the following Frontmatter at the top:

```yaml
---
title: My Delicious Recipe
type: recipe
image: ./photo.jpg  # Optional
servings: 4
total_time: 30
ingredients:
  - item: Pasta
    qty: 500
    unit: g
  - item: Tomato Sauce
    qty: 1
    unit: jar
steps:
  - time: 0
    action: Boil water.
  - time: 10
    action: Add pasta.
---
```
