# Animal Stack - Game Manual

## Overview
Stack the farm animals as high as you can without toppling the tower!

## How to Play
*   **Objective**: Create a stable tower of animals.
*   **Game Over**: If any animal falls off the screen, the game ends.

## Controls
*   **Arrow Left / Right**: Move current animal.
*   **Z / X**: Rotate animal.
*   **Space**: Drop animal.
*   **Touch**: Drag to move, tap bottom of screen to drop.

## Rules
*   Animals have different shapes and sizes.
*   Physics apply! Balance is key.
