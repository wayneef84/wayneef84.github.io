# Flow Games

A collection of flow-based puzzle games where players connect matching colors with pipes.

## Games Included

1.  **Flow Free**: Connect matching colors on a standard grid.
2.  **Flow Bridge**: Pipes can cross each other at specific bridge cells.
3.  **Flow Hex**: Connect colors on a hexagonal grid.
4.  **Flow Warp**: The grid wraps around the edges.

## Getting Started

Open `index.html` in a modern web browser.
