# SKYbreakers - Game Manual

## Overview
A high-flying vertical scrolling game where you navigate through treacherous obstacles.

## How to Play
*   **Objective**: Fly as far as you can without crashing into pipes or the ground.
*   **Scoring**: Pass between pipes to score points.

## Controls
*   **Space / Arrow Up**: Jump / Flap wings.
*   **Mouse / Touch**: Tap to jump.

## Features
*   **High Score**: Your best score is saved locally.
*   **Dynamic Difficulty**: The game speeds up and gaps get tighter as you progress.
