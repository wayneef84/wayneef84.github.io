## How to Add Real Images

1. Take or find photos (recommended size: 800x600px or similar aspect ratio)
2. Name them exactly as shown above (e.g., `kenzie.jpg`)
3. Place them in this directory
4. Refresh the game - images will automatically load!

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Recommended Specs
- **Aspect Ratio:** 4:3 or 16:9
- **Size:** 800x600px (or larger, will auto-scale)
- **File Size:** Under 500KB for fast loading
