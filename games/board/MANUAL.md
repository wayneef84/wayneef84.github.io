# Board Game Arcade - User Manual

## Overview
A collection of classic board games playable in your browser.

## Available Games
*   **Chess**: The game of kings.
*   **Checkers**: Jump and capture.
*   **Connect 4**: Get four in a row.
*   **Tic Tac Toe**: Classic 3x3.
*   **Mancala**: Stone sowing game.
*   **Battleship**: Sink the enemy fleet.
*   **Gomoku**: 5 in a row (Go board).
*   **Othello (Reversi)**: Flip opponent's pieces.
*   **Xiangqi**: Chinese Chess.

## How to Play
1.  **Select Game**: Use the overlay menu to choose a game.
2.  **Select Mode**:
    *   **PvP**: Local multiplayer (Hotseat).
    *   **AI**: Play against the computer.
3.  **Controls**: Click/Tap to select and move pieces.

## Features
*   **Undo**: Revert your last move.
*   **Reset**: Restart the current game.
