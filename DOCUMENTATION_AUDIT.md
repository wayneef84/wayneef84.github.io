## PROCESS

### How to Mark a Game As Complete

1. Use `/admin/DOC_TEMPLATES/` to create 7-file suite
2. Fill in all required sections (no WIP)
3. Verify 3-5 KB target per file
4. Update this audit table (change вқҢ to вң…)
5. Commit with message: `docs: Add documentation suite for [game]`
6. Update AGENT.md in game folder with session notes

### How to Update

Each time a game/project is documented:
1. Mark files complete (вқҢ вҶ’ вң…)
2. Change Status to "COMPLETE"
3. Move Priority to next uncompleted game
4. Update "Current Status" section
5. Commit: `audit: Mark [game] as documented`
